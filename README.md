# Програма для роботи з прямокутниками на мові C++

## Опис проекту

Цей проект реалізує систему для роботи з геометричними фігурами - прямокутниками. Програма дозволяє користувачеві введення набору прямокутників, обробляти їх та сортувати за площею.

## Структурні компоненти

### Структура `point` (12-02.h, 12-02.cpp)
Представляє точку в двовимірному просторі з координатами `x` та `y`.

**Функціональність:**
- Конструктори: параметричний та без параметрів
- Методи доступу: `getX()`, `getY()` для отримання координат
- Методи запису: `setX()`, `setY()` для встановлення координат
- Перевантажені оператори `>>` та `<<` для введення/виведення

### Клас `rectangle` (12-02.h, 12-02.cpp)
Представляє прямокутник, який визначається двома точками: верхнім лівим та нижнім правим кутами.

**Функціональність:**
- Конструктори: параметричний та без параметрів
- Методи розрахунків:
  - `width()` - обчислює ширину прямокутника
  - `height()` - обчислює висоту прямокутника
  - `area()` - розраховує площу прямокутника
- Перевантажені оператори `>>` та `<<` для введення/виведення із інтерактивними повідомленнями українською мовою

### Основна програма (result-12-02.cpp)
Головний модуль, що реалізує логіку роботи програми.

**Функціональність:**
1. Запитує користувача про кількість прямокутників
2. Динамічно створює масив прямокутників
3. Отримує дані кожного прямокутника від користувача
4. Сортує прямокутники за площею (від меншої до більшої) за допомогою алгоритму сортування бульбашкою
5. Записує відсортовані результати у файл `Result.txt`
6. Звільняє виділену пам'ять

## Технічні особливості

- **Мова:** C++
- **Парадигма:** Об'єктно-орієнтоване програмування
- **Перевантаження операторів:** Реалізовано для введення/виведення даних
- **Динамічна пам'ять:** Використання `new` та `delete` для управління масивом прямокутників
- **Файловий І/О:** Запис результатів у текстовий файл

## Користування

1. Запустіть програму
2. Введіть кількість прямокутників
3. Для кожного прямокутника введіть координати верхнього лівого та нижнього правого кутів
4. Програма автоматично сортує прямокутники за площею
5. Результати зберігаються у файл `Result.txt`
